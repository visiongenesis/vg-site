## The five locked v7 moves

### 1. Living Tree demoted from hero to inline diagram

The Tree is too rich a metaphor to abandon but too soft for social compression. v7 moves it out of the hero. Two placements:
- As a small inset visual in the hero (top-right or beside the headline)
- As a full-render inline diagram in the "Model" section (annotated)
- As a watermark fragment (intertwined-trunks icon) in section dividers + footer

The Tree stays on the site. It loses the full-bleed real estate.

### 2. Ryan in oxblood = VG's blue-shirt equivalent

Lock Ryan in an **oxblood crewneck or sweater** as the recurring visual mark. Every site hero photo, every YouTube thumbnail, every IG post, every speaking deck. Direct gaze. Operator energy.

- Until R provides a real photo: hero uses a styled placeholder slot ("Ryan Dix — operator portrait in production")
- HeyGen avatar fallback: lock the avatar in the same oxblood sweater + same lighting + same framing. Use the avatar UNTIL real photo is available, then swap.
- The board was unanimous: human face + locked color = instant recognition.

### 3. Numbered VG Stack

Adopt the Martell Ventures "01-05" device. v7's stack:

- **01 Operators** — They hold the truth.
- **02 Builders** — They build and own.
- **03 Agents** — They multiply judgment.
- **04 Practices** — Operating systems installed inside real businesses.
- **05 Products** — Solve once. Sell to the next thousand.

This replaces the current "Tree explainer" three-pillar layout. Each stack item gets one scroll-section with a single claim, a supporting line, and an optional artifact (Living Tree fragment, product card, etc.).

### 4. Signature refrain — *"We don't sell tools. We install operating systems."*

5 of 5 substantive responses landed on this exact construct or a variant. 9 words. Owns one VG distinction. Thumbnail-readable. Ladders into the locked tagline ("The system that builds yours.").

Deployed:
- Hero subhead (replacing v6.1's longer paragraph)
- Footer above the contact block
- Brand panel (paired with or replacing "We've built the system that allows us to build yours.")
- Every YouTube intro/outro
- X / LinkedIn bios
- Section breaks (small mono caps)

### 5. Typography shift toward Inter ExtraBold

- **Cormorant Garamond italic burgundy** — restrict to ONE italicized word in hero ("systems") + brand panel. Drop from everywhere else.
- **Inter** — primary heading face, weights 600–800 (heavier than v6.1's 600–700). Hero hed in 700–800.
- **Source Serif 4** — body prose only.
- **IBM Plex Mono** — section numbers (01/02/03/04/05), eyebrows, watermarks.

Bolder, denser, more contrast. Brutalist editorial.
